# 소설 비교 에디터

3개 AI 결과를 나란히 비교하고 선택한 문장이나 전체 텍스트를 최종본에 모으는 정적 웹앱입니다.

## Supabase 테이블

```sql
create table if not exists sessions (
  id bigint generated by default as identity primary key,
  session_key text unique not null,
  gpt_text text default '',
  gem_text text default '',
  cla_text text default '',
  final_text text default '',
  draft_text text default '',
  summary_text text default '',
  memo_cards jsonb default '[]',
  idea_cards jsonb default '[]',
  updated_at timestamptz default now()
);

alter table sessions add column if not exists summary_text text default '';
alter table sessions add column if not exists memo_cards jsonb default '[]';
alter table sessions add column if not exists idea_cards jsonb default '[]';

alter table sessions enable row level security;

create policy "Allow public session reads"
on sessions for select
using (true);

create policy "Allow public session inserts"
on sessions for insert
with check (true);

create policy "Allow public session updates"
on sessions for update
using (true)
with check (true);
```

Supabase Realtime에서 `sessions` 테이블의 update 이벤트를 켜면 다른 기기와 실시간 동기화됩니다.

## Vercel 환경변수

Vercel 프로젝트에 아래 값을 등록하세요.

- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`

## 로컬 실행

```bash
vercel dev
```

첫 방문 시 `localStorage`에 세션 키가 생성됩니다. 다른 기기에서 같은 내용을 보려면 `?s=세션키`가 붙은 공유 URL로 접속하면 됩니다.
